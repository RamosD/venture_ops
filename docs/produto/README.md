# Documentação de produto — VentureOps AI

Esta pasta contém a documentação funcional, técnica e operacional
correspondente ao **estado aceite** da solução: como o VentureOps AI funciona
actualmente, e não o histórico das tentativas de execução.

Descreve, entre outros, arquitectura, modelo de dados, APIs e contratos,
autenticação e autorização, regras de negócio, configuração, deploy, operação,
segurança, recuperação, integrações, procedimentos de utilização e limitações
conhecidas.

O registo de *como* a solução foi executada (pipelines, prompts, decisões,
riscos e validações) reside em `docs/gestao/`. Esta pasta é actualizada apenas
quando uma alteração aceite afecta o comportamento ou o desenho da solução.
